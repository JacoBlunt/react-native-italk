1.使用步骤
--------
    1.1：先clone到本地，相信这一步大家做开发的都基本会,不去详细描述
    1.2：然后进入到项目根目录 CMD命令执行 npm install 这一步用来加载些项目依赖主要通过读取配置文件package.json
         来加载大家不用细究执行就是了
    1.3: 然后在执行相应的平台命令
            Android执行react-native run-android
            IOS执行    react-native run-ios
